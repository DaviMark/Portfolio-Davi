Quem é Davi Marques?

Engenheiro de dados e Desenvolvedor Power BI com 7 anos de experiência no desenho de arquiteturas de dados escaláveis e na aceleração da maturidade analítica em grandes organizações. Minha atuação une a robustez da engenharia (pipelines, automação e governança) à inteligência de negócio, transformando ecossistemas de dados complexos em motores de decisão estratégica.
Atuo no desenho de arquiteturas de dados escaláveis e na criação de soluções de Business Intelligence que reduzem tempo de análise, aumentam a confiabilidade dos indicadores e fortalecem a cultura data-driven nas organizações.

Tenho domínio em SQL, Python e Cloud (BigQuery, Databricks, Oracle, SQL Server, MySQL e SAP HANA), aplicando práticas de governança, automação e otimização de pipelines ETL/ELT para garantir dados consistentes, acessíveis e seguros.
Entrego soluções que eliminam tarefas manuais, aceleram a tomada de decisão e ampliam a performance de áreas-chave como finanças, operações e marketing.

Dia 10 de Novembro/2025

Você já usou HTML dentro do Power BI? Em conversas com alguns analistas e desenvolvedores, percebi um movimento cada vez maior: o uso de visuais HTML dentro do Power BI. E foi aí que uma ideia me bateu: se no front-end temos bibliotecas como Bootstrap, Tailwind CSS e muitas outras, por que não criar uma biblioteca específica de visuais HTML para o Power BI (mas uma versão mini delas 😄)? Quem trabalha com dados sabe que em um dashboard uma das principais funções é gerar insights para Gestores, Coordenadores e lideres em geral para tomada de decisão ou compreender casos, mas com o passar do tempo e até para desenvolver uma cultura data-driven percebemos que não basta funcionar, TEM QUE IMPRESSIONAR! Gerar insights é essencial, mas criar uma experiência visual que prenda a atenção do gestor faz toda a diferença. Com essa proposta, nasceu o "UI Power Pack", que está disponível no link do primeiro comentario! É uma coleção de componentes HTML e CSS prontos para você inserir nos seus dashboards e transformar completamente o visual dos seus painéis. Hoje já são 30+ visuais, entre cards, tooltips, labels e documentações, e tem muito mais por vir. Também deixei alguns exemplos mostrando como um simples card pode se transformar em um card impressionante, além de uma forma criativa de incluir aquela breve explicação sobre o nosso querido (e polêmico) “Exportar para o Excel”. Uma breve demonstração do que podemos construir com esse conhecimento. E incluí uma página explicando o passo a passo de como implementar os códigos no Power BI e uma área para quem quiser colaborar com o projeto. Basta enviar seu componente pelo formulário e você aparece como criador do visual na galeria. O objetivo é ajudar a comunidade de dados a entregar dashboards que comuniquem melhor, inspirem mais e tenham aquele toque de “uau”.
