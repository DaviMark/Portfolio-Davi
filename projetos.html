Dashboard completo para gestão de estoque, oferecendo insights sobre faturamento, valor total em estoque, rupturas, giro de produtos e tendências ao longo do tempo. Visualize detalhes por fornecedor e área de armazenagem para otimizar a operação e a tomada de decisões.

Dashboard Logística

Dashboard logístico para análise completa da operação, incluindo malha de programação, custo total, valor de frete ao longo do tempo e OTIF. Acompanhe o total de viagens, distribuição por tipo de veículo, desempenho por região, além de métricas de cubagem e peso para otimização do transporte.

Dashboard Executivo

Envio de Email automatico

Este projeto é uma API desenvolvida com Node.js e Express que se conecta a um banco de dados SQL Server para recuperar informações financeiras e armazená-las em cache. Além disso, o sistema possui uma funcionalidade de envio automatizado de relatórios via email, utilizando Nodemailer e node-cron para agendamentos periódicos.

Abrir Visualizar

ETL de Dados

Este projeto consiste em um script Python que automatiza a inserção de dados em uma planilha Excel compartilhada. O usuário pode selecionar um arquivo Excel contendo informações, escolher algumas informações específica e inserir os dados filtrados no arquivo de destino.

Abrir Visualizar

Assistente Virtual Inteligente

Chatbot Inteligente, desenvolvido para otimizar o suporte interno da empresa, oferecendo respostas rápidas e precisas sobre processos, diretrizes e outras dúvidas operacionais.

Abrir Visualizar

Torre de Controle

O projeto Torre de Controle é uma plataforma desenvolvida para monitoramento e gestão de frota, integrando dados de rastreamento veicular, informações sobre motoristas, abastecimento, filiais e manutenção. O sistema permite uma visualização detalhada das operações, proporcionando uma gestão mais eficiente e baseada em dados atualizados.
