# Portfólio

Este é o meu portfólio profissional, desenvolvido utilizando **HTML, CSS, JavaScript e PHP**. O objetivo deste projeto é apresentar minhas habilidades, experiências e serviços prestados na área de **Análise de Dados, Desenvolvimento Web e Automação de Processos**.

## Sobre o Projeto

O portfólio inclui:
- Uma seção "Sobre Mim" com um breve resumo da minha trajetória profissional.
- Minhas principais habilidades técnicas.
- Um resumo da minha carreira, incluindo certificados de cursos e formação acadêmica.
- Exibição de projetos desenvolvidos, como dashboards, chatbots, conversores de arquivos e automação de processos.
- Uma seção de serviços oferecidos, abrangendo **Análise de Dados, Desenvolvimento Web e de Aplicativos, e Automação de Processos**.
- Uma área de contato com informações como telefone e e-mail.

## Tecnologias Utilizadas
- **Front-end:** HTML, CSS, JavaScript
- **Back-end:** PHP

## Funcionalidades
- Apresentação interativa dos projetos desenvolvidos.
- Seção de contato para fácil comunicação.
- Layout responsivo para melhor experiência em dispositivos móveis.
